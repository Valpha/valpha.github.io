学生管理系统的要求

  • 利用数据库实现对 学生 { “姓名”、“学号” } 进行管理
  • 需要能实现 增加、删除、修改、查询 四个功能
    • 输入学生 { “姓名”、“学号” } 信息可以添加到数据库
    • 给出“姓名”或“学号”或“姓名”+“学号”来删除已有的数据信息
    • 给出“姓名”和“学号”信息，将符合学号的数据的姓名修改为给出的姓名
    • 查询所有存在的数据信息
    • 给定筛选条件，按照指定的“姓名”或“学号”或“姓名”+“学号”来查询数据信息

关键技术点

操作接口

  • 两个输入文本框：{ 姓名、*学号 * }
  • 四个按钮：{ 插入、删除、修改、查询 }
  • 一个输出列表：( {姓名、学号}*n )
