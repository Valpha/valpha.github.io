SimpleAdapter的使用

设置Adapter

五个参数：

  • 上下文——this

  • 数据本体——data

  • 布局——R.layout.item1

  • 数据来源的字段——from

  • 对应的控件——to

  • ```java
    lv = (ListView)findViewById(R.id.lv_);
    final List<Map<String, String>> data = new ArrayList<>();
    for (int i = 0; i < 10; i++){

    Map<String, String> map = new HashMap<>();
map.put("Name", "zhangsan"+i);
map.put("PhoneNumber", "1001000100"+i);
data.add(map);

    }
    String[] from = new String[]{“Name”, “PhoneNumber”};
    int[] to = new int[] {R.id.tv_name, R.id.tv_phonenumber};
    SimpleAdapter adapter = new SimpleAdapter(this,

        //  数据本体
    data,
    //  布局
    R.layout.item1,
    //  数据的字段
    from,
    //  数据对应的位置
    to);

    lv.setAdapter(adapter);
